# MealNChill Progressive Web App (PWA)

MealNChill is now a Progressive Web App! This means you can install it on your mobile device and use it like a native app.

## Features

### 📱 Mobile Installation

- **Android**: Install directly from Chrome browser
- **iOS**: Add to Home Screen from Safari
- **Desktop**: Install from Chrome, Edge, or other supported browsers

### 🌐 Offline Support

- App works even when you're offline
- Automatic background sync when connection is restored
- Cached pages for faster loading

### 🔄 Auto Updates

- App updates automatically in the background
- Always get the latest features without manual updates

## How to Install

### On Android (Chrome)

1. Open MealNChill in Chrome
2. Tap the "Install" prompt that appears at the bottom
3. Or tap the menu (⋮) → "Add to Home Screen"
4. The app will appear in your app drawer

### On iOS (Safari)

1. Open MealNChill in Safari
2. Tap the Share button (□↗)
3. Scroll down and tap "Add to Home Screen"
4. Tap "Add" to confirm
5. The app will appear on your home screen

### On Desktop (Chrome/Edge)

1. Open MealNChill in your browser
2. Look for the install icon in the address bar
3. Click "Install" when prompted
4. The app will open in its own window

## Troubleshooting

### App Not Installing?

- Make sure you're using a supported browser
- Check that you have enough storage space
- Try refreshing the page and attempting again

### Offline Mode Not Working?

- The app needs to be visited at least once while online
- Some features require internet connectivity
- Clear browser cache and reload if issues persist

### App Not Updating?

- Close and reopen the app
- Clear browser cache if needed
- Updates happen automatically in the background
